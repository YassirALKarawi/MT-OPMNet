# Contributing to MT-OPMNet

Thanks for your interest in improving this project.

## Current Scope

This repository is currently in a documentation-first research snapshot stage. The full implementation is planned for a later release.

At this stage, the most useful contributions are:

- documentation improvements,
- issue triage and clarification,
- reproducibility and packaging suggestions.

## Before You Open an Issue

Please include:

1. a clear title,
2. expected behavior,
3. observed behavior,
4. any relevant references (paper section, figure, or equation).

## Pull Request Guidelines

- Keep changes focused and atomic.
- Use clear commit messages in imperative tense.
- Update documentation when behavior or structure changes.
- Avoid adding large binaries to the repository.

## Style Guidelines

- Use readable Markdown headings and short sections.
- Prefer explicit wording over implied assumptions.
- Keep terminology consistent (e.g., OSNR, MFI, AAH, CAAM).

## Code of Conduct

By participating, you agree to maintain a respectful and constructive environment.
